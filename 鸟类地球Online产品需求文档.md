## 0\. 项目总览（AI 全局前置认知）

本项目为**纯静态鸟类科普自适应官网**，无后端、无数据库、无SPA、无前端路由，基于 HTML + Tailwind CSS + 原生JavaScript 开发。全站支持 PC/手机双端完美响应式适配，核心能力包含：科普图文展示、多图图集预览、B站视频自适应播放、全局柔和动效、首页水滴环形动态交互、前端JSON模糊搜索、模态框弹窗系统。整体为**清新趣味极简科普UI**，风格轻盈柔和、层级干净、交互治愈，面向大众鸟类科普场景。

## 1\. 全局设计规范（强制统一，全站遵循）

### 1.1 视觉风格定位

Soft Neumorphism 2.0轻悬浮质感 + 趣味极简SaaS风 + 亲和现代UI + 分层矢量插画；低刺激、低对比、高柔和，无尖锐边角、无刺眼高饱和色块，整体轻量化高级。

### 1.2 固定色彩系统

* 基础背景：纯白
* 主题辅助色：薄荷绿 #84DB5D、天空蓝 #1C9DE4
* 视觉规则：仅小面积点缀使用主题色，整体低对比柔和，支持半透明渐变背景

### 1.3 排版规范

* 标题：粗体几何无衬线字体，层级醒目，可搭配薄荷绿细下划线点缀
* 正文/按钮/辅助文字：圆角无衬线亲和字体（Nunito）
* 排版层级：超大标题 / 中等正文 / 小号辅助文字，层级清晰明确

### 1.4 造型与阴影规范

* 全站所有组件统一**大圆角设计**，无任何尖锐边角
* 全局采用低透明度、高模糊柔化投影，实现轻悬浮3D质感
* 功能图标极简统一，识别度高、风格贴合自然科普调性

### 1.5 全局动效规范（强制）

* 所有交互动效使用 **柔和弹性缓动曲线 cubic-bezier 弹性值**
* 页面全局支持滚动渐显、轻微视差效果
* 页面、组件切换为低对比柔和渐变过渡，无生硬跳转
* 可交互元素默认自带轻微悬浮微动效，整体轻盈治愈

## 2\. 技术架构与开发约束（不可修改）

### 2.1 技术栈

HTML + Tailwind CSS + 原生JavaScript，不引入任何重型框架、不使用Vue/React、不使用前端路由。

### 2.2 项目架构（多页面静态架构）

全站仅3个核心HTML文件，无SPA，详情页单文件复用

* index.html 首页
* list.html 鸟类图鉴卡片页
* detail.html 鸟类详情通用页（通过 URL ?id= 参数动态渲染数据）

### 2.3 数据方案（强制）

* 数据源：独立静态文件 `/data/bird\\.json`，**禁止JS硬编码数据**
* 读取方式：原生fetch异步读取，全站共用一套数据结构
* 数据量级：适配几十\~几百条数据，支持长期迭代新增修改
* 本地调试要求：使用Live Server运行，无需兼容file://协议

### 2.4 标准目录结构

### 2.5 全局路由跳转规则

* 首页水滴点击 → 弹窗预览后可跳转 list.html 图鉴页
* 图鉴卡片按钮 → 跳转 detail.html?id=对应鸟类ID
* 首页搜索框 → 关键词匹配后直接跳转对应详情页，无结果给出文字提示

## 4\. 页面详细需求（逐页严格实现）

### 4.1 首页 index.html（核心重点页面）

#### 4.1.1 页面布局

* 顶部：通用吸顶导航栏
* 主体严格 **1:2 左右分栏**
* 左侧1/3：固定展示网站名称「鸟类地球Online」
* 右侧2/3：水滴环形鸟类动画区域

#### 4.1.2 水滴环形视觉规则（核心动态UI）

* 圆心位置：右侧2/3区域 水平居中、垂直居中
* 圈层结构：共4圈环形，**内圈最小、外圈最大**
* 水滴形态：每颗水滴尖端朝向圆心、圆润弧面朝外，花瓣式环绕
* 基础动效：4圈水滴**同转速、匀速缓慢持续环绕旋转**
* 固定规则：页面滚动时，环形整体位置固定不动
* 资源方案：现阶段使用emoji+CSS动画实现，预留后续SVG/rive替换接口

#### 4.1.3 PC端交互（支持Hover）

* 常态：4圈水滴持续匀速旋转
* Hover单颗水滴：该水滴放大1.5倍+高亮柔阴影；**当前圈层暂停旋转，其他圈层继续旋转**，不改变水滴朝向和环形结构
* 鼠标移出：水滴恢复原尺寸，对应圈层恢复旋转
* 点击水滴：全局旋转缓动停止 → 水滴先向圆心收拢、再向外弹性弹出 → 唤起全屏高斯模糊蒙版+居中预览模态卡片
* 弹窗内容：鸟类大图、简短文字介绍、关闭按钮、查看图鉴按钮
* 弹窗操作：关闭后恢复首页环形动效；点击查看图鉴跳转 list.html

#### 4.1.4 移动端专属交互（无Hover，两级Tap）

* 基础适配：外圈水滴点击热区 ≥44px，适配触屏点击精度
* 常态：保留完整4圈缓慢旋转动效，屏幕自适应缩放
* 一级交互（轻触Tap）：点击水滴触发1.5倍放大回弹弹性反馈，直接跳转对应鸟类详情页 detail.html?id=xxx
* 二级交互（长按Hold）：长按暂停环形自动旋转，可手动滑动旋转圆环，松开后恢复自动环绕旋转

### 4.2 图鉴卡片页 list.html

* 入场动效：从首页水滴点击位置执行 **easing弹性展开入场动画**
* 布局：自适应网格，PC多列、手机单列，无布局错乱
* 卡片样式：大圆角、柔阴影、轻悬浮质感
* 卡片Hover交互：背景高斯模糊+轻微上浮动效
* 卡片功能：内置查看详情按钮，点击跳转对应详情页
* 全局动效：页面滚动渐显、轻微视差

### 4.3 鸟类详情页 detail.html（通用复用页）

根据URL参数id读取bird.json数据，动态渲染所有内容

* 页面结构：鸟类标题、科普大图、详细科普文字、多图图集、基础参数介绍、B站视频播放模块
* 图集交互：任意图片点击唤起模态框，实现大图预览，支持蒙版/按钮关闭
* 视频模块：16:9自适应B站iframe，支持缩略图弹窗播放
* 全局动效：滚动渐显、轻微视差，所有按钮/标签统一柔和hover样式
